Данное приложение принимает сообщения (статистики по ТС) 
в виде массивов байт из Очереди сообщений или по протоколу Http,
преобразовывает их в объект используемого класса 
и добавляет в базу данных.

Для настройки метода получения сообщений необходимо 
изменить соответвующие параметры в файле app.config.
Если при запуске приложения в аргументах командной строки
передаётся метод получения сообщений, то приоритетным будет он.
