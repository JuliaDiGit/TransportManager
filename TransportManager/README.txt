Приложение TransportManager используется для работой с базой фирм, 
их водителей и их транспортными стредствами. 
Проект "Statistics" обрабатывает информацию из бд 
и составляет отчёты о поездках ТС. 

Приложение MessageSenderEmulator отправляет сообщения 
с информацией о поездках ТС.

Приложение TelemetryStorageServer принимает сообщения 
и записывает полученную информацию в бд.

Для настройки запуска приложений TelemetryStorageServer и MessageSenderEmulator
необходимо изменить соответвующие параметры в файле app.config, 
по умолчанию запуск модулей отключён.
Перед тем, как запускать данные модули вместе с текущим приложением - 
необходимо предварительно их собрать.
